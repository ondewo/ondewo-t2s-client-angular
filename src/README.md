# @ondewo/t2s-client-angular

`@ondewo/t2s-client-angular` is a compiled version of the [ONDEWO T2S API](https://github.com/ondewo/ondewo-t2s-api).

## Installation

Using NPM:

```shell
npm i --save @ondewo/t2s-client-angular
```
